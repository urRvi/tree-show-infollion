# React Flow Tree Visualizer

A client-side interactive tree visualizer built with [React Flow](https://reactflow.dev/) and Vite. This project implements a recursive layout algorithm to display hierarchical data with expand/collapse functionality, as part of the Infollion candidate intake assignment.

- **Interactive Nodes**

- **Deterministic Re-layout**

- **Client-Side Only**  
  Fully browser-based implementation with no backend dependencies, as specified in the assignment.
